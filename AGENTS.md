## Cursor Cloud specific instructions

This repository is a **GitHub profile README** (`README.md` only). There is no application code, no dependencies, no build system, and no tests.

- **No services to run.** The repo contains a single `README.md` with HTML-formatted content (shields.io badges, social links, bio).
- **No lint/test/build commands.** There are no package managers, lock files, or configuration files.
- **Preview:** To visually verify changes to `README.md`, serve the repo with a static HTTP server (e.g., `python3 -m http.server 8080`) and open the file in a browser, or use the GitHub preview directly.
- **Editing:** The README uses raw HTML (not Markdown syntax). Badges are hotlinked from `img.shields.io` and other CDNs.
